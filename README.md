# PES Version Control System (Mini Git)

## Name: Sanjith PES2UG24CS445

## Description
This project is a simplified version of Git implemented in C.
It supports basic version control operations like init, add, commit, log, branch, and checkout.

## Features Implemented

### Phase 1
- Initialize repository (`pes init`)

### Phase 2
- Add files to index (`pes add`)

### Phase 3
- Commit changes (`pes commit -m "message"`)

### Phase 4
- View commit history (`pes log`)

### Phase 5
- Create branch (`pes branch <name>`)
- Switch branch (`pes checkout <name>`)

## How to Run

```bash
make
./pes init
./pes add file1.txt
./pes commit -m "first commit"
./pes log
